This Project Repository contains 4 jupyternotebook (.ipynb) files and 3 folders:

ipynbs:
	Sample Run.ipynb:
		Contains code for the algorithm and an execution example.
	Simulated Experiments.ipynb:
		Contains code, experiment results and result analysis for Experiments which use simulated data stream.
	Real_world.ipynb:
		Contains code, experiment results and result analysis for Experiments which use Original real world data stream.
	Real_world-mix.ipynb:
		Contains code, experiment results and result analysis for Experiments which use Periodical real world data stream.
Folders:
	configs: Experiment configeritions
	Raw_data: Real World Data Stream
	results : Experiment results
